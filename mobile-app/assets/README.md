# Assets Directory

This directory should contain the following image assets:

## Required Files

### icon.png
- App icon displayed on the device home screen
- Size: 1024x1024px
- Format: PNG with transparency
- Design: Use the AI Model Platform logo with dark background (`#0f1115`)

### splash.png
- Splash screen shown during app load
- Size: 1284x2778px (iPhone) or similar aspect ratio
- Format: PNG
- Design: Centered logo on dark background (`#0f1115`) with subtle gradient

### adaptive-icon.png (Android)
- Foreground layer for Android adaptive icons
- Size: 1024x1024px
- Format: PNG with transparency
- Design: Logo centered, allowing for safe zone cropping

### favicon.png (Web)
- Browser favicon for web builds
- Size: 48x48px
- Format: PNG

## Generating Assets

You can use tools like:
- Figma or Sketch for design
- [Expo Icon Builder](https://buildicon.netlify.app/) for generating all sizes
- ImageMagick for batch resizing

## Color Scheme

- Background: `#0f1115`
- Primary accent: `#2dd4bf`
- Secondary accent: `#0ea5e9`
